# Serene

Serene est une application mobile simple (Ionic) pour gérer des sessions de méditation avec minuteries.

Prérequis
- Node.js
- npm
- Ionic CLI (optionnel) : `npm install -g @ionic/cli`

Installation
```
npm install
```

Lancer en développement
```
ionic serve
```

Construire pour la production
```
npm run build
```
-
Technologies
- Ionic Framework (Ionic + Angular)
- Angular (framework web)
- TypeScript
- SCSS (styles)
- Node.js et npm (outils de développement)

Fichiers importants
- Le code source se trouve sous `src/` (pages, composants, styles).
